# 🚀 Design 1 – RPO Landing Page

A modern SaaS-style RPO (Recruitment Process Outsourcing) landing page built using Next.js, TypeScript, and Tailwind CSS.

Clean UI. Responsive layout. Component-based architecture.

---

## 🔗 Link

https://design-1-iota.vercel.app/

---

## 🛠 Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Bun / Node.js

---

## ✨ Features

- Modern Hero Section with CTA  
- RPO Services Section  
- Reusable Components  
- Fully Responsive Design  
- Clean Folder Structure  

---

## 🎯 Purpose

This project was built to practice modern SaaS-level UI design using Next.js and TypeScript while following clean and scalable frontend architecture principles.
